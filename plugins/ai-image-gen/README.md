# ✨ AI Image Generation / AI 生图

Generate images using AI models with multi-model support.

使用 AI 模型生成图片，支持多模型配置。

## Supported Providers / 支持的服务商

- Google Gemini
- SiliconFlow / 硅基流动
- OpenRouter
- Volcano (Doubao) / 火山引擎 (豆包)
- Custom API / 自定义 API

## Usage / 使用方法

1. Configure API Key in extension settings / 在扩展设置中配置 API Key
2. Right-click on canvas -> 'AI Generate Image' / 右键画布 -> 'AI 生成图片'
3. Or right-click on a node -> 'Generate Image' to use node text as prompt / 或右键节点 -> '生成图片' 使用节点文本作为提示词
4. Use the image generation panel for more options / 使用生图面板可选择更多选项

## Requirements / 要求

- MeowThink >= 0.56.0
- Valid API Key for chosen provider / 所选服务商的有效 API Key
